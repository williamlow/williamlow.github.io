Data and performance

Conflict events are sourced from ACLED data. SC Nigeria Early Warning System data will be used when available.

Model performance is measured by running large scale simulations of past conflicts and verifying predictions against IOM DTM data. The current model has a 16% error rate.

Notes on use

These results should be used to assess likely future trends in displacement patterns. Figures should not be treated as exact outcomes, and should always be used in concert with other situation assessment and decision-making processes.
